# linux 终端快捷键

# 基本快捷键

ctrl+b 左

ctrl+f 右

ctrl+a 光标移动到行首

ctrl+e 光标移动到行尾

ctrl+u 删除光标至开始全部内容

ctrl+k 删除光标至末尾全部内容

ctrl+w 删除光标前单词

ctrl+y 粘贴使用

# tab键的妙用

作用：补全路径 补全命令 可能需要用的命令和路径

用法：单机tab 双击tab

# 文件权限介绍

'r' 可读

'w' 可写

'x' 可执行/切换场景

'-' 无权限

# 通配符

'*' 匹配0个或者多个字符

'?' 匹配一个字符

[abcd] 配置abcd中间任意一个字符

[a-z] 匹配a-z范围的任意一个字符

[!abc] 不匹配放括号里的任何一个字符
