# Linux

# linux 终端快捷键

# 基本快捷键

ctrl+b 左

ctrl+f 右

ctrl+a 光标移动到行首

ctrl+e 光标移动到行尾

ctrl+u 删除光标至开始全部内容

ctrl+k 删除光标至末尾全部内容

ctrl+w 删除光标前单词

ctrl+y 粘贴使用

# tab键的妙用

作用：补全路径 补全命令 可能需要用的命令和路径

用法：单机tab 双击tab

# 文件权限介绍

'r' 可读

'w' 可写

'x' 可执行/切换场景

'-' 无权限

# 通配符

'*' 匹配0个或者多个字符

'?' 匹配一个字符

[abcd] 配置abcd中间任意一个字符

[a-z] 匹配a-z范围的任意一个字符

[!abc] 不匹配放括号里的任何一个字符

# man 命令

# 常用快捷键

空格 下一页

回车键 下滚一行

j 下滚一行

k 上滚一行

b 上一页

f 下一页

q 退出

/关键字 搜索 n 下一个关键字 shift+n上一个关键字

# 参数

-f 显示关键字的简短描述信息

-k 更具关键字搜索帮助手册

-w 显示手册文件所在的位置

# touch 命令

创建文件

  1. 创建一个文件

touch hsj.txt

  1. 创建多个文件

touch hsj.txt zy.txt

# mkdir 命令

创建文件夹

# 参数

-p 递归创建多级目录

-m 建立目录的同时设置权限

-v 显示目录的创建过程

# 例子

  1. 创建多个目录

mkdir hsj zy

  1. 递归创建目录

mkdir -p hsj zy

# rm 命令

删除文件或者文件夹

# 参数

-f 忽略不存在的文件,不会出现警告信息

-i 删除前询问用户是否删除

-r/-R 递归删除

-v 显示指令的详细执行过程

# 例子

  1. 删除一个文件

rm file.txt

  1. 删除一个目录

rm -r dir

  1. 强制删除文件/目录 不带提示

rm -f file.txt

  1. 删除当前目录下所有的文件

rm -rf*

  1. 删除前逐一询问确认 rm -i file1.txt file2.txt

# mv 移动文件夹

1、移动文件 2、重命令文件

# 参数

-i 若存在同文件名，则向用户询问是否覆盖

-f 覆盖已有文件时，不进行任何提示

-b 当文件存在时，覆盖前为其创建一个备份

-u 当源文件比目标文件新，或者目标文件不存在时，才执行移动此操作

# 案例

  1. 将文件 file.txt 移动到 dir 文件夹中

mv file.txt dir

  1. 将file.txt文件 重命名 为 newfile.txt

mv file.txt newfile.txt

  1. 将 目录dir1 移动到目录 dir2中 （前提是 dir2目录存在 不存在则重命名）

mv dir1 dir2

  1. 将 file.txt 重命名为 newfile.txt 如果newfile.txt 已经存在，则询问是否覆盖

mv -i file.txt newfile.txt

  1. 将file.txt 重命名为 newfile.txt 如果newfile.txt 已经存在，则直接覆盖

mv -f file.txt newfile.txt

  1. 文件覆盖前做好备份

mv -b file.txt newfile.txt

  1. 源文件比目标文件新时才执行更新

mv -u file.txt newfile.txt

  1. 将当前文件夹下所有内容移动到上一层目录

mv * ..

  1. 将当前目录的一个子目录里的文件移动到另一个子目录里面

mv dir2/* dir3
