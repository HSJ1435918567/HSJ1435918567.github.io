# 常用注解

# java 元注解

# @Retention

@Retention 是一个元注解，其作用可以简单理解为设置注解的声明周期。 Retention 注解传入的是 RetentionPolicy 枚举，该枚举有三个常量，分别是 SOURCE、CLASS 和 RUNTIME

  • SOURCE 代表着注解仅保留源级别中，编译器将 Java 文件编译成 class 文件时将之遗弃

  • CLASS 代表着注解被保留咋 class 文件中，JVM 加载 class 文件时将之遗弃

  • RUNTIME 代表着标记的注解会由 JVM 保留，因此运行环境可以使用它

# @Documented

注解表明这个注解应该被 javadoc 工具记录. 默认情况下,javadoc 是不包括注解的. 但如果声明注解时指定了 @Documented,则它会被 javadoc 之类的工具处理, 所以注解类型信息也会被包括在生成的文档中，是一个标记注解，没有成员。

  • IDEA 生成 JavaDoc

# @Target

使用@Target 元注解进行标注，用它来指定当前注解可以出现在哪些位置上；

  • @Target(ElementType.METHOD) 方法

  • @Target(ElementType.TYPE) 接口、类、枚举、注解

  • @Target(ElementType.FIELD) 字段、枚举的常量

  • @Target(ElementType.PARAMETER) 方法参数

  • @Target(ElementType.CONSTRUCTOR) 构造函数

  • @Target(ElementType.VARIABLE) 局部变量

  • @Target(ElementType.ANNOTATION——TYPE) 注解

  • @Target(ElementType.ANNOTATION——PACKAGE) 包，用于记录 Java 文件的 package 信息

# @Inherited

表示其标记的注解允许被继承，接口实现除外
